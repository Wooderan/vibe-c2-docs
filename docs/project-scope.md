# Project Scope

## Vision

Build a reliable, operator-friendly C2 platform with strong observability and security controls.

## In Scope (Phase 1)

- Core teamserver capabilities.
- Operator authentication and role-based access.
- Agent-to-server secure communication channel.
- Basic tasking, result collection, and audit logging.
- Documentation-first technical planning.

## Out of Scope (Phase 1)

- Full plugin marketplace.
- Advanced offensive modules beyond baseline testing workflows.
- Enterprise multi-tenant architecture.

## Stakeholders

- Product owner
- Security engineering
- Backend/platform engineering
- Operator users

## Success Criteria

- Critical workflows are documented and agreed.
- Technical requirements reviewed and version-controlled.
- Architecture baseline supports planned MVP scope.
